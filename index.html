¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un modelo de diseño CSS que permite crear layouts flexibles y eficientes sin usar floats o positioning. Facilita la distribución del espacio y la alineación de elementos, incluso cuando sus tamaños son desconocidos o dinámicos.

Ejes y Direcciones

Mi explicación: El primero es el eje principal, que define hacia dónde se colocan los elementos dentro del contenedor, y se controla con la propiedad flex-direction. Puede ser en forma de fila (row), columna (column), o en orden inverso (row-reverse, column-reverse).


Cómo influye: Transforma instantáneamente un contenedor normal en uno flexible, haciendo que todos sus hijos se acomoden en una fila por defecto y permitiendo usar todas las demás propiedades de Flexbox.

Alineación de elementos

Mi explicación: Flexbox permite la alineacion de elementos dentro del contenedor y en esta hay dos propiedades importantes que son: Justify-content que sirve para alinear los elementos en el eje principal. Por ejemplo, se pueden colocar al inicio, al centro o al final. También se pueden repartir con espacios iguales entre ellos (space-between, space-around, space-evenly).


Cómo influye: Me permite crear espacios perfectos entre elementos sin calcular márgenes manualmente. Con space-between hago que el primer elemento esté pegado a la izquierda y el último a la derecha automáticamente. ¡Adiós a los márgenes complicados!


Archivo 2: Alineación de elementos

Alineación de elementos multilínea

Mi explicación: Cuando el espacio no alcanza para poner todos los elementos en una sola línea, se puede usar la propiedad flex-wrap, que hace que los elementos se acomoden en varias líneas. Después, con la propiedad align-content, se controla cómo se distribuyen esas líneas completas dentro del contenedor.


Cómo influye: Hace que mis diseños sean verdaderamente responsivos. En una pantalla grande tengo 5 elementos en una fila, en el celular se reorganizan solos en 2 o 3 filas. El navegador hace todo el trabajo de decidir cuándo "envolver" los elementos a la siguiente línea, también me da control sobre el espacio vertical cuando tengo varias filas. Puedo hacer que las filas estén pegadas arriba, centradas verticalmente, o distribuidas uniformemente.


Archivo 3: Alineación de elementos multilínea

Alineación individual

Mi explicación: A veces queremos que un solo elemento esté alineado de forma diferente al resto. Para eso se usa align-self, que permite modificar la posición de un elemento en el eje transversal sin afectar a los demás. Por ejemplo, si todos los elementos están centrados, pero quieres que uno de ellos se coloque abajo, puedes usar align-self: flex-end; solo en ese elemento.


Cómo influye: Esto da más control y flexibilidad al diseño, porque cada componente puede tener su propio comportamiento. Puedo tener un logo centrado mientras otros elementos están alineados arriba, o hacer que un botón de "Enviar" esté en una posición especial sin crear un contenedor separado.


Archivo 4: Alineación individual

Propiedades de los elementos hijos

Mi explicación: permite controlar cómo cada elemento hijo se adapta al espacio disponible dentro del contenedor. Las propiedades principales son: flex-grow: define cuánto puede crecer un elemento si hay espacio libre. flex-shrink: define cuánto puede reducirse si el espacio es pequeño. flex-basis: establece el tamaño base del elemento antes de repartir el espacio.


Cómo influye: Me permite crear layouts fluidos donde unos elementos son más importantes que otros. Por ejemplo, en un layout de 3 columnas, puedo hacer que la columna central (el contenido) crezca y ocupe todo el espacio disponible, mientras los sidebars mantienen un tamaño fijo.

Conclusión

Flexbox es una herramienta poderosa que simplifica enormemente el diseño web. Con solo unas pocas propiedades, puedo crear layouts complejos, responsivos y visualmente atractivos sin complicaciones.
